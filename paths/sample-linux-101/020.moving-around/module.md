# Moving around

The single most used command in any shell is `cd` (change directory). This
module trains it until it becomes muscle memory: going down into
directories, jumping back up, and returning home from anywhere.

A few facts worth keeping in mind while you do your reps:

- `cd somedir` goes down into `somedir`
- `cd ..` goes one level up
- `cd` with no arguments goes to your home directory from anywhere
- `cd -` returns to the previous directory
